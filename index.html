Real-time object recognition of ship videos in Edge Device environment

In collaboration with Avikus, we developed a real-time object recognition system for ship videos in an Edge Device environment. Our system identifies and tracks objects of interest, such as vessel, motorboat, navigation sign, or sailing yacht to improve safety and situational awareness for ship operators.

Understanding Interpreters and Implementing Tibero's Functionality

This project is structured into three distinct sections, each focusing on a specific aspect. The first section delves into the implementation of an Interpreter using the Go programming language. Moving forward, the project addresses the practical implementation of several key functionalities within Tibero. The initial focus lies on Materialized views, followed by the detailed implementation of Tibero's TABLE_TO_COMMA and COMMA_TO_TABLE features, which are facilitated by the DBMS_UTILITY package. Lastly, the project explores the development of UTL_SMTP within Tibero, a crucial package utilized for email transmission via the Simple Mail Transfer Protocol (SMTP).
